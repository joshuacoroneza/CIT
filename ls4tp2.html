Format Specifiers


In C programming we need lots of specifiers to work with various data types. Format specifiers define the type of data to be printed on standard output. Whether to be printed on standard output or to take formatted input we need format specifiers. Format specifiers are also called as format string.
Here is a complete list of all format specifiers used in C programming language.
