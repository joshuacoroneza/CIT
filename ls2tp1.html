SYMBOLIC CONSTANT

Objectives : At the end of the lesson students are expected to:

  • Define symbolic constants
  • Explain how DEFINE is used in C programming
  • Apply the concept of DEFINE in C programming

A symbolic constant is name that substitute for a sequence of character that cannot be changed. The character may represent a numeric constant, a character constant, or a string. When the program is compiled, each occurrence of a symbolic constant is replaced by its corresponding character sequence. They are usually defined at the beginning of the program. The symbolic constants may then appear later in the program in place of the numeric constants, character constants, etc., that the symbolic constants represent.
Example
,aC Program consists of the following symbolic constant definitions.
#define PI 3.141593
#define TRUE 1
#define FALSE 0
#define PI 3.141593 defines a symbolic constant PI whose value is 3.141593. When the program is pre-processed, all occurrences of the symbolic constant PI are replaced with the replacement text 3.141593.
Note: The preprocessor statements begin with a # symbol and are not end with a semicolon. By convention, preprocessor constants are written in UPPERCASE.
